# Dataplotter documentation

This documentation follows the [Diataxis](https://diataxis.fr/) framework. Use the right document type for your goal.

| Goal | Document type | Use when |
|------|----------------|----------|
| **Learn** the project and run it the first time | [Tutorials](tutorials/) | You're new; you want a step-by-step path. |
| **Do** a specific task | [How-to guides](how-to-guides/) | You have a concrete problem (e.g. add a graph type, add an LLM). |
| **Look up** facts (APIs, env vars, structure) | [Reference](reference/) | You need a technical description or key facts. |
| **Understand** why things are the way they are | [Explanation](explanation/) | You want background, trade-offs, or concepts. |
